# Zephyr scope

The Zephyr app is the firmware-side port of the Arduino portal MVP. It should match the same setup and dashboard flow before adding sensor-specific code.

Out of scope for this first Zephyr pass:

- MR60BHA2 sensor parsing
- OTA update
- Cloud sync
- Native mobile app
- Desktop app

In scope:

- Setup AP
- Captive DNS
- HTTP setup pages
- Wi-Fi scan/connect
- Saved credentials
- Local dashboard page
